# **App Name**: HealthBridge AI

## Core Features:

- Role Selection Landing Page: Displays cards for different user roles (Citizen, ASHA Worker, Clinical Login, Admin/Govt) on the landing page.
- Login Modal: Opens a modal upon clicking a role card, allowing users to input User ID/Mobile and Password, with a role dropdown pre-filled based on the selected card.
- Role-based Navigation: Redirects users to specific dashboard pages (/citizen/dashboard, /asha/dashboard, /clinical/dashboard, /admin/dashboard) based on their selected role upon successful login.
- Citizen Dashboard: Displays health records, symptom checker button, and appointment booking functionality.
- ASHA Worker Dashboard: Provides village tracking, immunization schedules, and a maternal care visit list.
- Clinical Dashboard: Enables OPD management, viewing lab reports, and managing referrals.
- Admin Dashboard: Shows analytics charts/cards, infrastructure monitoring, and audit logs.

## Style Guidelines:

- Primary color: Dark blue (#1E3A8A) to convey trust and professionalism.
- Background color: Light blue (#E3F2FD) a subtle variant of the primary color to establish trust.
- Accent color: Sky blue (#60A5FA) provides interactive elements while not overwhelming the user.
- Body and headline font: 'Plus Jakarta Sans' sans-serif. Note: currently only Google Fonts are supported.
- Cards with a glass effect and hover animations to provide a modern and interactive feel.
- Smooth transitions for modal openings and role-based navigation for enhanced user experience.
- Fully responsive layout, optimized for mobile devices to ensure accessibility on all screen sizes.
